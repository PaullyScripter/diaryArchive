# DiaryArchive Admin Skill

Build tools for administrators, not end users.

Include:
- report queue
- reported users
- reported diaries
- reported comments
- audit logs
- storage usage
- search indexing health
- system metrics

Admins can never decrypt private diaries.
Moderation is manual after user reports.
