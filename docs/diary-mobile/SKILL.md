# DiaryArchive Mobile Skill

Use:
- React Native
- TypeScript
- Expo (unless native modules require otherwise)

Goals:
- share backend APIs with web
- offline drafts
- secure token storage
- responsive UX
- accessibility
- push notifications
- media uploads

Keep feature parity with the web app where practical.
